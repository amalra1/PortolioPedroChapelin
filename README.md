## HTML

- Geralmente o nome da página principal, por convenção, é **index.html**
- O HTML é visto por *tags*, ou seja, tudo para ser representado deve ser assim:  
  ```html
  <tag>
    (conteúdo)
  </tag>
  ```
**--> Tags**

- `<h1>`, `<h2>`, ..., `<h6>` --> Heading (cabeçalhos);
- `<p>` --> Parágrafo
